# 🍳 CookApp - 食谱手册

**CookApp** 是一款基于 Android 的食谱管理应用，采用 MVC 架构设计，支持用户登录、食谱浏览、收藏、购物清单等多种功能，界面简洁美观，操作流畅。

---

## 📱 主要功能

- 用户注册 / 登录 / 修改密码
- 食谱列表浏览、筛选、收藏、分享
- 购物清单管理（添加 / 删除）
- 持久化数据存储（基于 Room 数据库）
